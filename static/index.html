⚠ IMPORTANTE
El formato CORPAC no incluye el tipo de NOTAM en el texto. Para identificar si es NOTAM N (nuevo) o NOTAM R (reemplaza), consulte el boletín original en extranet.corpac.gob.pe o pegue el texto en formato estándar OACI donde el tipo sí aparece explícito (ej: C4854/25 NOTAMR C3568/25).
